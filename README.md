## 👨‍🏫 프로젝트 소개

라이엇 API를 활용해 LOL 챔피언과 아이템, 로테이션 챔피언에 대한 정보를 볼 수 있는 사이트

## 💜 주요기능

- 모든 챔피언 목록

- 모든 챔피언 상세 페이지

- 모든 아이템 목록

- 금주의 로테이션 챔피언

## ⏲️ 개발기간

- 2024.09.25(수) ~ 2024.10.08(화)
